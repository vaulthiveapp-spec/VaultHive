# VaultHive Font Assets

Place Poppins TrueType font files here. Download from:
https://fonts.google.com/specimen/Poppins

Required files:
- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf
- Poppins-ExtraBold.ttf
- Poppins-Black.ttf

After adding the files, rebuild the app with `eas build` or `expo start`.

The app renders with the OS system font if these files are absent.
